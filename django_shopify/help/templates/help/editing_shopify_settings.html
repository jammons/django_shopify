Editing Your shopify_settings.py

Next up you have to put the API Key and Shared Key Values into the shopify_settings.py file.

  1. Open shopify_settings.py in the root of your Django repository.
  2. Copy the API Key and Shared Key values as SHOPIFY_API_KEY and SHOPIFY_API_SECRET. If you know what permission scope you need you can edit that as well.
