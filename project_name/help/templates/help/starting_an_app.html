Starting a Shopify App

The first thing you're going to need to do to get this Django app running is to create an app on Shopify. This is fairly simple:

  1. Create an account on app.shopify.com
  2. Create an app and enter http://localhost:8000 for the application URL.
  3. Copy your API Key and Shared Secret into the shopify_settings.py file. (You have to click "show secret" to see it.
